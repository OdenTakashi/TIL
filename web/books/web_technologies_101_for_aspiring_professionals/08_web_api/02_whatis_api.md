## API(Application Programming Interface)とはなにか
コンピューターが互いにコミュニケーションを取るための取り決め

部品ごとに分けるメリットはなにか
個々の部品ごとに分けることで下記のメリットがある
- 製品効率の向上
  - 分担することで早く作れるようになる
- 品質向上
  - 分担することで部品の品質向上に集中できる
- 交換しやすい
  - 部品単位での修理、交換が可能になる

## ソフトウェアの部品化とAPI
printf のようにAPIとしての呼び出しが決まっているため、中で起きていることを無視できる
下記が決められていることで内部実装を知らなくても問題がない
- 呼び出すための名前
- パラメーター
- 戻り値
